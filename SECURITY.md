# 🛡️ Security Policy: Project Nebula

## 📜 FreshAssassin's Code of Chaos
Project Nebula is designed strictly for **educational, testing, and harmless pranking purposes.** While the tool provides significant control over the browser's Document Object Model (DOM), it is engineered to be **100% Client-Side**. This means:
* It does NOT interact with server-side databases.
* It does NOT steal cookies, passwords, or personal data.
* It does NOT persist after a page refresh (F5).

## ⚠️ Responsible Usage
By using this tool, you agree to use it ethically. Do not use Project Nebula to harass, deceive for financial gain, or disrupt critical infrastructure. 

## 🕵️‍♂️ Reporting a Vulnerability
If you find a bug in the code that could be used maliciously or causes a permanent browser crash, please do not open a public issue. Instead:
1. Contact the lead developer (**FreshAssassin**) via GitHub private messaging or the designated contact point.
2. Provide a detailed description of the vulnerability.
3. Allow time for a patch to be issued before public disclosure.

**Keep the chaos fun. Keep the web safe.**
